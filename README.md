# FlavorLoop - Static HTML Website

A modern, visually rich USA-based recipe website built with pure HTML, CSS, and JavaScript.

## üìÅ File Structure

### HTML Pages
- **index.html** - Home page with hero section, featured recipes, categories, and newsletter
- **recipes.html** - All recipes page with search and filtering functionality
- **recipe-detail.html** - Individual recipe detail page with ingredients and instructions
- **about.html** - About Us page with mission, values, and story
- **contact.html** - Contact page with form and contact information
- **terms.html** - Terms & Conditions page
- **privacy.html** - Privacy Policy page
- **404.html** - 404 Error page

### CSS Files
- **styles.css** - Complete stylesheet with responsive design, all page layouts, and components

### JavaScript Files
- **script.js** - Main JavaScript for navigation, toast notifications, and newsletter
- **recipes-data.js** - All recipe data (12 recipes)
- **recipes-page.js** - Recipes page functionality (search, filter, display)
- **recipe-detail.js** - Recipe detail page functionality
